<objective>
The DE-VEGAS materiality workstream — TP3's non-blocking perceptual re-authoring, applied ON TOP of the
instanced clay body from 04-02 (operator-locked sequence: de-Vegas AFTER instancing). Re-author the chips
as matte worn clay that RECEDES: kill the gloss (body clearcoat 0.42->0.32, clearcoatRoughness 0.46->0.5,
sheen 0.5->0; face clearcoatRoughness 0.36->0.5), tool the C/rim as a RECESSED mark by swapping the face
bumpMap for a normalMap via the shared normalMapHelper, mute the accent chroma ~20% + lower value so the
chips recede behind the cards, desaturate + shrink the face logo, and add restrained micro-grain. Behind
a `?chips=` flag for apples-to-apples A/B.

most-saturated object after cards) and read as worn artisanal clay (C tooled-not-printed, no Vegas gloss),
NOT plastic. This is NON-BLOCKING: if the chips read plastic or pull the eye, STOP and revert (keep the
instanced-but-pre-de-Vegas chips — instancing already shipped in 04-02). The operator A/B gate is a
SEPARATE plan (04-04). No heavy central pile (the pot stays demoted); inter-chip AO is TP6, not here.

Output: a de-Vegas chip body + face material (clearcoat 0.32 / clearcoatRoughness 0.5 / sheen 0; face
normalMap), muted palette + desat/shrunk logo + micro-grain in textures.ts, behind a `?chips=` A/B flag,
with a recorded M2 + recede + MACRO-quality verdict (ship) OR a non-blocking revert.
