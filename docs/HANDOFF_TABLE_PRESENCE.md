## Slices

### L1 ‚Äî Ambient felt breathe ("el tapete respira") ‚Äî committed `17bbe53`
- **What:** a dedicated `.table-breath` layer behind the felt (`z-index:-1`, centre occluded by the opaque felt) whose **opacity** breathes on a slow 10s cycle, gently deepening the table's resting drop shadow. Weight + atmosphere, **never motion**.
- **Why this mechanic:** the first try animated `box-shadow` directly ‚Üí desktop ~42fps (repaint). Recalibrated to opacity-only (compositor-cheap) ‚Üí **~60fps desktop + mobile**. The felt's own `box-shadow` stays constant.
- **Files:** `frontend/index.html` (`<div class="table-breath">` first child of `.table-surface`) + `frontend/src/style.css` (`.table-breath` rule + `@keyframes table-breathe` + `prefers-reduced-motion` guard).
- **Runtime gate (PASS):** felt shadow constant; breath opacity 0‚Üí1; ~60fps both; reduced-motion pins it off; vitest 319/319; build clean.
- **Perceptual status:** **DIRECTION APPROVED by operator** ("la firma perceptual es correcta"). Amplitude is deliberately subliminal (`maxDelta ~1.1/255` rest‚Üípeak). On-device amplitude fine-tune **optional/open**.
- **Amplitude knob:** `.table-breath { box-shadow: 0 40px 110px rgba(0,0,0, 0.12) }` ‚Äî the `0.12` alpha is the depth of the breath.

### M1 ‚Äî Woven baize (felt fibre) ‚Äî committed `e125176`
- **What:** a faint, **static**, desaturated‚Üí**green-tinted** fractal-noise fibre tiled over `.table-surface` and composited at low alpha, so the green reads as woven cloth ("el verde parece pa√±o").
- **Why this mechanic (important for resume):** `soft-light` looked like cloth but **lightened the dark felt** (fibre and lightening were coupled). Root cause: SVG filters run in **linearRGB** by default (green came out brighter) + soft-light is asymmetric on dark. Fix: **mean-neutral green-on-green** tint (same idea as the `::after` micro-variation) + `color-interpolation-filters='sRGB'`. This **preserves the green AND decouples fibre amount from tone**.
- **Files:** `frontend/src/style.css` ‚Äî `.table-surface` `background-image` now has 2 layers (noise + the unchanged felt gradient) with `background-size`/`background-repeat`. No blend-mode (normal). The felt gradient/centre/wood/gold-rim are unchanged.
- **Runtime gate (PASS, desktop + Pixel 5):** colour meanLum Œî **+0.4** (same green); fibre stddev **0.43‚Üí1.15** (present); maxŒî **~5** (subtle); diff = fine random fibre (**no pattern / no tile seams**); build clean; vitest **319/319**.
- **Perceptual status:** **PENDING ‚Äî operator on-device gate not done.** Conservative amplitude; subliminal at real size, visible as fine fibre at 3x zoom.
- **Amplitude knob (now decoupled from colour):** `.table-surface` weave `rect opacity` (currently **0.30**) + the `feFunc*` slopes. Raise opacity to ~0.38‚Äì0.40 for more presence; the tint is felt-centred so the colour stays put.

atmosphere, NOT more FX*. The phrase for M1 is **"el verde ahora parece pa√±o."**

**Hard NOs (operator-locked):** casino / Vegas / crypto-casino ¬∑ glow ¬∑ spotlight ¬∑ neon ¬∑ gaming UI ¬∑

mobile-first ¬∑ `prefers-reduced-motion` ¬∑ identity invariants (felt gradient, wood 16px, gold rim,
